# Задание B1: "Простой скрипт-помощник" (Bash)

## Цель
Написать bash-скрипт для очистки старых лог-файлов

## Скрипт: clean_old_logs.sh

### Функциональность
1. Принимает два аргумента: путь к директории и количество дней (N)
2. Находит все файлы с расширением .log старше N дней
3. Выводит список файлов с информацией (размер, дата изменения)
4. Запрашивает подтверждение на удаление
5. При 'y' - удаляет файлы, при 'n' - отменяет
6. Выводит справку при неправильном использовании

### Использование
```bash
./clean_old_logs.sh /path/to/logs N
```
### Ключевые особенности скрипта
1. Проверка существования директории

2. Валидация входных параметров

3. Подробный вывод информации о файлах

4. Интерактивное подтверждение удаления

5. Защита от случайного удаления
